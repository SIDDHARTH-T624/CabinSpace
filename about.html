About My Cabin Space

My Cabin Space is a simple web-based cabin/seat selection system designed to simulate a real-world booking interface. The project allows users to visually view available, occupied, and selected cabins in an intuitive grid layout.

How It Works

  • Cabins are divided into AC and Non-AC sections for clarity.
  • Each cabin is represented as a selectable unit with a unique label.
  • Color indicators show the current state of each cabin: Available, Selected, and Occupied.
  • Users can select only one cabin at a time, ensuring clear and controlled selection.
  • The interface updates the selection visually without page reloads.

Use Case / Purpose

This project demonstrates frontend fundamentals such as structured HTML, responsive CSS, and basic JavaScript interaction, focusing on clarity, usability, and clean UI behavior.
